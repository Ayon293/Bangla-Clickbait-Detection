# Bangla Clickbait Detection using BanglaBERT and Machine Learning

A Natural Language Processing (NLP) project that detects Bangla clickbait headlines using **BanglaBERT**, **TF-IDF**, and Machine Learning models.

---

## 📖 Project Overview

Clickbait headlines are designed to attract readers through sensational or misleading language. This project develops an automated binary classification system to identify whether a Bangla news headline is **Clickbait** or **Non-Clickbait**.


---

## ✨ Features

* Bangla text preprocessing
* BanglaBERT embedding extraction
* TF-IDF feature extraction
* Handcrafted text features
* Multiple Machine Learning classifiers
* Model comparison
* Performance visualization
* Confusion Matrix generation
